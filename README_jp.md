MinecraftMapImager
======================

MinecraftMapImagerはmap_\*.datファイルから画像への変換プログラムです。


動作環境
-----------

* Java 8 (JRE8 / JDK 1.8) 以降


使い方
-----------

`$ java -jar mapImager.jar [-o OUTPUT_FILE] SOURCE ...`

デフォルトの出力ファイル名は`map.png`です。

WindowsではmapImager.exeにmap_～.datファイルをドラッグ＆ドロップすることでも動作します。
